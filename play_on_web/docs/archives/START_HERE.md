# 👋 从这里开始！

欢迎使用 XLerobot Web Teleop！

## 🎯 您只需要 3 步

### Step 1️⃣: 验证环境
```bash
cd /Users/ai/Project/play_lerobot/play_on_web
./verify_setup.sh
```

如果看到 ✅ 表示环境配置正确。

### Step 2️⃣: 启动服务
```bash
./start_conda.sh
```

等待服务启动...

### Step 3️⃣: 打开浏览器
```
http://localhost:3000
```

就这么简单！ 🎉
