## 🔖 快速链接

### 常用命令

```bash
# 验证环境
./verify_setup.sh

# 启动服务（Conda）
./start_conda.sh

# 启动服务（Venv）
./start.sh

# 检查 Python 依赖
conda activate lerobot
cd backend
python check_env.py
```

### 常用地址

- 前端: http://localhost:3000
- 后端: http://localhost:8000
- API 文档: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc
