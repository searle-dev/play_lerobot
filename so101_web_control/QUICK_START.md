# 快速开始指南

## 🚀 一键启动

```bash
cd /Users/ai/Project/play_lerobot/so101_web_control
bash start.sh
```

或者直接运行：

```bash
cd /Users/ai/Project/play_lerobot/so101_web_control
python app.py
```

然后在浏览器中打开：**http://localhost:5000**

## 📋 操作步骤

### 1. 启动服务器

在终端中运行上面的命令，看到类似输出：

```
============================================================
SO101 机械臂网页控制服务器
============================================================
访问地址: http://localhost:5000
============================================================
 * Running on http://0.0.0.0:5000
```

### 2. 打开网页

在浏览器（Chrome/Safari/Firefox）中访问：`http://localhost:5000`

### 3. 连接机器人

1. 确认端口正确（默认：`/dev/tty.usbmodem5A7C1163141`）
2. 点击"连接"按钮
3. 等待状态变为"已连接"（绿色）

### 4. 开始控制！

#### 方式 A：键盘控制
直接按键盘上的键：
- `Q/A` → 关节1
- `W/S` → 关节2
- `E/D` → 关节3
- `R/F` → 关节4
- `T/G` → 关节5
- `Y/H` → 夹爪

#### 方式 B：鼠标点击
点击界面上的虚拟按键

## ✨ 特色功能

- 🎮 **双重控制**：键盘 + 虚拟按键
- 📊 **实时反馈**：实时显示关节当前位置和目标位置
- ⚙️ **可调参数**：步进大小、夹爪步进、比例增益
- 📝 **操作日志**：记录所有操作
- 🔄 **一键重置**：快速回到零位

## ⚠️ 注意事项

1. **首次使用**：确保机器人已经在终端中成功校准过
2. **端口确认**：如果连接失败，用 `ls /dev/tty.usb*` 确认端口
3. **安全第一**：确保手臂周围有足够空间
4. **紧急停止**：在终端按 Ctrl+C 停止服务器

## 🔧 故障排查

### 问题1：连接失败
```bash
# 检查端口
ls /dev/tty.usb*

# 测试连接
cd /Users/ai/Project/lerobot/examples/xlerobot
python3 -c "from lerobot.robots.so100_follower import SO100Follower, SO100FollowerConfig; robot = SO100Follower(SO100FollowerConfig(port='/dev/tty.usbmodem5A7C1163141')); robot.bus.connect(); print('连接成功'); robot.bus.disconnect()"
```

### 问题2：网页打不开
- 检查服务器是否正常启动
- 尝试 `http://127.0.0.1:5000`
- 检查防火墙设置

### 问题3：夹爪不动
- 确认夹爪已正确校准
- 尝试调大"夹爪步进"参数

## 🌐 远程访问

如果想从其他设备访问（手机/平板）：

1. 确认所有设备在同一局域网
2. 在 Mac 上运行 `ifconfig | grep "inet "` 查看 IP
3. 在其他设备浏览器中访问：`http://[您的IP]:5000`

例如：`http://192.168.1.100:5000`
