## 🎮 快速操作指南

### 步骤 1：连接机器人
1. 确认端口：`/dev/tty.usbmodem5A7C1163141`
2. 点击"连接"按钮
3. 等待绿色"已连接"状态

### 步骤 2：开始控制
**键盘控制：**
- `Q/A` → 底座旋转 ← →
- `W/S` → 大臂升降 ↓ ↑
- `E/D` → 小臂弯直 ⌒ ⌐
- `R/F` → 腕部上下 ↓ ↑
- `T/G` → 腕部旋转 ↺ ↻
- `Y/H` → 夹爪开关 ◄ ►

**或点击虚拟按键！**

### 步骤 3：调节参数（可选）
- **步进大小**：5° （推荐）
- **夹爪步进**：10 （推荐）
- **比例增益**：0.5 （推荐）

## 🛠️ 故障排查速查

| 问题 | 解决方案 |
|------|---------|
| 连接失败 | `ls /dev/tty.usb*` 检查端口 |
| 网页打不开 | 检查服务器是否启动 |
| 夹爪不动 | 确认已校准，调大步进 |
| 控制延迟 | 降低频率，调整 Kp |

详细说明见 [QUICK_START.md](QUICK_START.md)

## 💡 小贴士

1. **首次使用**：建议小步进测试（设为 2-3°）
2. **频繁操作**：可增大步进到 10-15°
3. **精细控制**：降低步进到 1-2°
4. **手机访问**：更适合虚拟按键操作
5. **紧急停止**：终端按 Ctrl+C
